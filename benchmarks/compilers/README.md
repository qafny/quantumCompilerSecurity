# Quantum Compilers Benchmark Collection

This directory contains information and resources about major quantum compilers for benchmarking and comparison purposes.

## Compiler List

### Core Compilers (7)

1. **[Qiskit](qiskit.md)** - IBM's quantum software development framework
2. **[Cirq](cirq.md)** - Google's quantum computing framework
3. **[Silq](silq.md)** - High-level quantum programming language (ETH Zurich)
4. **[VOQC](voqc.md)** - Verified Optimizer for Quantum Circuits
5. **[ScaffCC](scaffcc.md)** - Scaffold quantum compiler (UC Berkeley)
6. **[PennyLane](pennylane.md)** - Quantum machine learning framework with compiler
7. **[Quipper](quipper.md)** - Functional quantum programming language
